# Proyecto Comunicaciones WeCel
## Proyecto de programación I
### Realizado por: 
Julian Andres Gomez Solano  
Juan Jose Molina Chaparro.
Estudiantes de Ingeniería de Sistemas y Computación– UPTC 

Comunicaciones WeCel es una proyecto desarrollado en Java que implementa una arquitectura basada en capas (Entidades, Servicios, Controladores y Excepciones) para la gestión y simulación de un sistema de comunicación.
El proyecto fue diseñado con buenas prácticas de programación orientada a objetos, manejo de excepciones personalizadas y documentación JavaDoc generada automáticamente.

### Tecnologías utilizadas:

Java 17+
Maven (gestor de dependencias y build)
JavaDoc (documentación del código)
POO (Programación Orientada a Objetos)

### Documentación JavaDoc

El proyecto cuenta con documentación generada mediante JavaDoc, donde se describen las clases, métodos y excepciones utilizadas.
Puedes acceder al archivo principal de la documentación aquí:
/doc/index.html

O ver documentación JavaDoc en línea aca:
https://newjuqn.github.io/Proyecto-Comunicaciones-WeCel/ 

### Ejecución del proyecto

Clona el repositorio:
git clone https://github.com/NewJuqn/Proyecto-Comunicaciones-WeCel.git
Abre el proyecto en tu IDE (Eclipse, IntelliJ, VS Code, etc.).
Compila y ejecuta con Maven:
mvn clean install
